數字累加器(支援複數以上)#

範例#

const sum = (...arr) => [...arr].reduce((acc, val) => acc + val, 0);
sum(1, 2, 3, 4); // 10
sum(...[1, 2, 3, 4]); // 10

解說#

...arr // 採用函式的其餘參數，函式內會是以陣列接收。
[...arr] // 其餘參數的陣列透過解構為一個新陣列

跑 reduce 累加器

reduce((acc, val) => acc + val, 0);
// acc 是 accumulator 上一次呼叫累加的總數值
// val 是 currentValue 當前要累加的數值
// 後面傳入 0 為 initialValue 累加器初始值
